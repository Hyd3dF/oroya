# Oroya — Web Sitesi

Oroya markası için tek sayfalık profesyonel kurumsal web sitesi.

## Dosya Yapısı

- `index.html` — Tüm içerik (hero, hakkımızda, hizmetler, fiyatlandırma, neden oroya, iletişim, footer)
- `style.css` — Tüm stiller (CSS custom properties, dark theme, responsive)
- `script.js` — Nav scroll, mobile menu, scroll animations, iletişim formu mailto
- `robots.txt` — SEO: arama motoru izinleri
- `sitemap.xml` — SEO: sayfa haritası
- `photo.jpeg` — Hero fotoğrafı (attached_assets'ten kopyalandı)

## Teknik Detaylar

- **Stack:** Saf HTML + CSS + JavaScript (framework yok)
- **Sunucu:** static-web-server, port 80
- **Tema:** Koyu (dark), Apple tarzı minimal premium
- **Renkler:** `#0a0a0a` bg, `#d4b896` warm accent
- **Font:** System font stack (-apple-system)
- **SEO:** meta tags, OG tags, JSON-LD schema, robots.txt, sitemap.xml
- **Responsive:** Mobile-first, 3 breakpoint (900px, 640px, 400px)
